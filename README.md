## Para rodar o projeto:

## 1 - Instalar as dependências

```
cd backend
```

```
npm install
```

## 2 - É necessário que o backend esteja rodando.

```
cd backend
```

```
node index.js
```

## 3 - Rodar a landing page do projeto.

```
cd frontend
```

```
node script.js
```

## 4 - Digitar a palavra a ser buscada no formulário e clicar "buscar".
